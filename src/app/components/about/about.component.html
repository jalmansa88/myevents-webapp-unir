Trabajo Final de Máster - 2018

Web App de la Aplicación MyEvents para la gestión de imágenes de Eventos.

El alta de los usuario se realiza a través de Tokens generados por los administradores de cada evento, asignando un evento y un nivel de permiso para el registro usando dicho token.

Cada usuario tiene un nivel de permiso (rol) por cada evento. Éstos son:

  • Nivel 0: Usario anónimo. Sólo puede ver las imágenes públicas de un único evento. No permite registro.
  • Nivel 1: Usario básico. Permite crear una cuenta y registrarse a múltiples eventos.
  • Nivel 2: Usario VIP. Tiene acceso a las imágenes privadas.
  • Nivel 3: Usario Administrador. Gestiona los usuarios y las imágenes del evento.
  • Nivel 4: Proveedor del Servicio. Crea los eventos pero no tiene acceso a ellos. Es el responsable de generar tokens para registros de nuevos administradores.

Los usuarios también podrán acceder a las imágenes a través de la aplicación móvil de MyEvents
